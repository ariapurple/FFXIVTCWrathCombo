# WrathCombo - Dalamud API Level 12

A downgraded version of WrathCombo plugin for Dalamud API Level 12.

## Installation

### Method 1: Custom Repository (Recommended)
1. In FFXIV with Dalamud, type `/xlsettings`
2. Go to "Experimental" tab
3. Add this repository URL:
   ```
   https://raw.githubusercontent.com/ariapurple/FFXIVTCWrathCombo/main/repo.json
   ```
4. Go to "Plugin Installer" and search for "Wrath Combo"
5. Install the plugin

### Method 2: Manual Installation
1. Download the latest release from [Releases](https://github.com/ariapurple/FFXIVTCWrathCombo/releases)
2. Extract `WrathCombo.zip` to your Dalamud plugins folder
3. Restart FFXIV

## Features
Condenses combos and mutually exclusive abilities onto a single button - and then some.
